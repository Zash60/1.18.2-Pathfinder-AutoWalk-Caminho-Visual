# Minecraft 1.18.2 Pathfinder AutoWalk

Mod para Minecraft Java 1.18.2 com:
- Pathfinding e navegação automática
- Auto-walk toggle (tecla O)
- Caminho visual renderizado NOS BLOCOS (nível do pé)
- Comando `/goto <x> <y> <z>`
- Salto automático sobre obstáculos

## Build
Baixe o JAR compilado em: **Actions > Build Mod > Artifacts**

## Uso
- `/goto 100 64 -200` - Navega até as coordenadas
- `P` - Ativa/Desativa pathfinder
- `O` - Ativa/Desativa autowalk simples
